# AI Workforce Productivity & Burnout Risk Analysis

## Project Overview

This project analyzes workforce productivity, burnout risk, and automation adoption using SQL and Power BI.

The objective was to understand how AI adoption, manual workload, and deadline pressure influence employee wellbeing and operational performance.

---

## Tools Used

- MySQL (Data exploration & validation)
- Power BI (Dashboard Development)
- SQL
- DAX

---

## Key Business Questions

- Which job roles experience the highest burnout?
- Does AI usage reduce manual workload?
- How does deadline pressure impact performance?
- Which teams show operational risk patterns?

## Key Insights

- Managers demonstrate higher burnout levels compared to other roles.
- Increased deadline pressure correlates with increased burnout risk.
- AI adoption reduces manual workload in several roles.
- Workforce risk patterns vary significantly across roles.
